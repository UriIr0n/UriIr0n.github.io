# תיק עבודות BI — אתר אינטראקטיבי

אתר סטטי (HTML/CSS/JS ונילה, ללא ספריות) שמציג 5 שחזורים אינטראקטיביים של דשבורדים
שנבנו במקור ב-Power BI — עם **נתוני דמה בדיוניים** לשמירה על פרטיות הארגון.
נבנה עם **Claude Code**.

## צפייה מקומית
פשוט לפתוח את `index.html` בדפדפן (דאבל-קליק). אין צורך בשרת.

## העלאה לאוויר (בחינם) — כדי לשתף בלינקדאין

### אפשרות א' — GitHub Pages (מומלץ)
1. פתחו חשבון ב-github.com (אם אין)
2. New Repository ← שם: `bi-portfolio` ← Public ← Create
3. לחצו "uploading an existing file" וגררו את **כל הקבצים שבתיקייה הזו** (כולל תיקיות css ו-js)
4. Settings ← Pages ← Branch: `main` ← Save
5. תוך דקה-שתיים האתר חי בכתובת: `https://<שם-המשתמש>.github.io/bi-portfolio/`

### אפשרות ב' — Netlify Drop (הכי מהיר)
1. גשו אל app.netlify.com/drop
2. גררו את התיקייה `portfolio-site` כולה לדף
3. מקבלים כתובת מיידית (אפשר לשנות לשם ידידותי בהגדרות)

## מבנה
```
index.html          עמוד ראשי + ניתוב לדשבורדים
css/style.css       כל העיצוב (RTL, דארק מוד)
js/charts.js        מנוע תרשימי SVG (בר, עמודות, דונאט, קווים, נערם, טבלה)
js/shell.js         שלד אחיד לעמוד דשבורד (כותרת, טאבים, "על הפרויקט", סיפור)
js/data.js          מחולל נתוני הדמה (דטרמיניסטי — אותם נתונים בכל טעינה)
js/dash-*.js        חמשת הדשבורדים
js/main.js          כרטיסי פרויקטים + ניתוב + רספונסיביות
```
